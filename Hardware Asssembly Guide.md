# HiWonder TurboPi Assembly Guide & Correct Video Order

The official HiWonder YouTube playlist for the TurboPi assembly contains videos that are out of order, along with several feature showcases mixed in. This can make it challenging to figure out the exact assembly sequence. 

Below is the verified, step-by-step assembly order with notes to help you avoid common mistakes during the build.

---

## 1. Check Contents
Before beginning, unbox all components and verify that you have all the necessary parts, screws, and tools listed in your kit.

## 2. Attach the Expansion Board to the Raspberry Pi
Securely mount the HiWonder expansion board onto your Raspberry Pi before attaching it to the vehicle chassis.

* **Optional Overview:** [HiWonder Pi Expansion Board Overview](https://youtu.be/k4s9Au3EF6Y?si=PgsI46sfaEnbTByz) — Useful for understanding the layout and ports of the expansion board.
* **Assembly Video:** [TurboPi Assembly: Assemble Raspberry Pi + Expansion Board](https://youtu.be/52N_oKo334s?si=4og4as8ZxAK10uI2)

## 3. Attach the Controller & Line Follower to the Main Body
Mount the combined Raspberry Pi and expansion board unit to the main body chassis, then install the tracking sensor.

* **Assembly Video:** [TurboPi Assembly: Install Controller + Line Follower](https://youtu.be/F5LvTlMeYEM?si=ZsP-zIZYwazmX7WW)
* ⚠️ **Crucial Note:** Ensure you complete this step, **including all relevant cabling**, before installing the motors and wheels. Failing to route cables first makes them highly inaccessible later.

## 4. Install Motors, Wheels, and Battery Holder
Once the internal controller and line follower are fully wired and secured, proceed to build out the drivetrain and power supply.

* **Assembly Video:** [TurboPi Assembly: Install Motors + Wheels + Battery Holder Case](https://youtu.be/erCFkrZ1qKg?si=nWRbYBGvE_l8QWKt)

## 5. Assemble the "Head" (Camera & Sensors)
Construct the upper sensor array, which includes the pan-tilt mechanism, camera, and ultrasonic sensor.

* **Assembly Video:** [TurboPi Assembly: Assemble Servo + Ultrasonic Sensor + Camera](https://youtu.be/46S2ZhOeooE?si=OvOc0oJ7FajFvmZj)
* 💡 **Tip:** If the horizontal servo feels loose or doesn't "stick" using only the mounting method shown in the video, use some of the included screw nuts to firmly clamp and lock it into place.

## 6. Attach the Head to the Body (Final Step)
Mount the completed pan-tilt camera head onto the main chassis and secure the protective expansion board cover.

* **Assembly Video:** [TurboPi Assembly: Assemble Pan Tilt + Expansion Board Cover](https://youtu.be/vKcvwtmXHYs?si=jLAC-dm6xtmcSF)
